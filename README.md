# 🎮 坦克大战 (Tank Battle)

经典的坦克大战（Battle City）网页版游戏，使用纯 HTML/CSS/JavaScript 实现。

## 🕹️ 玩法介绍

- **移动**: `↑` `↓` `←` `→` 或 `W` `A` `S` `D`
- **射击**: `空格键`
- **暂停**: `P`

## 🎯 游戏规则

- 消灭所有敌方坦克即可过关
- 保护底部的**基地**（鹰形标志），基地被摧毁则游戏结束
- 每关有 20 辆敌方坦克
- 击败敌人有概率掉落道具

## 🎨 特性

- 🎨 **Canvas 渲染** - 流畅的像素风格画面
- 🧱 **多种地形** - 砖墙（可摧毁）、钢铁（需升级）、水域（不可通过）、树林（隐藏效果）
- 🤖 **AI 敌人** - 4 种敌人类型：基本、快速、强力、装甲
- 💎 **道具系统** - 火力升级、额外生命、全屏炸弹、冻结敌人、临时无敌
- 📈 **难度递增** - 每关地图不同，敌人更强
- 💥 **屏幕震动** - 爆炸和摧毁效果
- ✨ **粒子特效** - 爆炸、砖块碎片等
- 📊 **侧边信息栏** - 实时显示分数、生命、剩余敌人

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

或者使用本地服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js (需要安装 http-server)
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`
